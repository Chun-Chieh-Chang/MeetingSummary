# Project Requirements: MeetingSummary Pro

## Goals
- High-fidelity meeting summarization using Google Gemini 1.5.
- Speaker diarization (who spoke when).
- Action item extraction with owners.
- Premium dark-mode UI with glassmorphism.

## Technical Constraints
- Frontend: React + TypeScript + Vite.
- Backend: Serverless (direct API calls from browser).
- Storage: Local (browser storage for history).

## UI/UX Standards
- Follow Digital Art Director guidelines.
- Responsive design for mobile and desktop.
- 1.5x line height for readability.

# Project Requirements (Entity)

This entity represents the core requirements and constraints defined for the `MeetingSummary Pro` project.

## 📝 Synthesis from [raw/requirements.md](../../raw/requirements.md)

### 1. Functional Scope
- **Audio Processing**: Direct browser-to-Gemini audio streaming.
- **Diarization**: Automated speaker labeling (Speaker 1, 2...).
- **Summarization**: Extraction of decisions and action items.

### 2. Technical Stack
- **Framework**: Vite + React (TypeScript).
- **AI Model**: Gemini 1.5 Pro/Flash.
- **State Management**: React Hooks (useState, useRef).

### 3. Design Aesthetics
- **Theme**: Premium Dark (Slate 900).
- **Style**: Glassmorphism (Backdrop blur + thin borders).
- **Typography**: Outfit/Inter font stacks.

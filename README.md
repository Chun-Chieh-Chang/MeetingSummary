# 🎙️ MeetingSummary

> A premium, agent-driven meeting summarization and task extraction system.

Built with **Antigravity** principles: Surgical modifications, Robustness testing, and Digital Art Director UI standards.

## 🧠 Project Brain (Wiki)
This project uses the [LLM Wiki Pattern](wiki/concepts/llm-wiki.md) to maintain a compounding knowledge base.
- [Wiki Index](wiki/index.md)
- [Governance & Schema](wiki/SCHEMA.md)

## 🚀 Vision
To transform chaotic meeting transcripts into elegant, actionable summaries using state-of-the-art LLM logic and a premium user experience.

## 🛠️ Tech Stack (Target)
- **Frontend**: React / Next.js (TailwindCSS for rapid, high-fidelity styling)
- **Logic**: Javascript/Typescript
- **Intelligence**: Advanced Agentic Summarization Logic

## 📂 Structure
- `wiki/`: Knowledge Base (Entities & Concepts).
- `raw/`: Source transcripts and meeting notes.
- `src/`: Application source code (TBD).
- `DEV_LOG.md`: Chronological development history (PDCA).
